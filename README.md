Hi, this is Sirja and Amelie and our final CS32 Project will be creating an Art Curator


Museum Curator:

1. Pulling out all the artwork information from the Harvard Art Museum and putting it into a big CSV file (is this too much? Should we condense?)
    a. For FP step 1, we’re just going to start by making a simple length 25 CSV file and working from there.
2. Write a program to find art pieces for gallery with certain keywords or patterns (can we tell what is inside the picture?), specifying:
    a. Date created (i.e., time duration)
    b. Artist
    c. Country of origin
    d. Material (Print, paint)
    e. Color (pull out Jpeg’s as well?)
    f. Dimensions
3. Curate - pull out 20 of the artworks that match your keyword/search terms the best
